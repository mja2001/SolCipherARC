## üé® User Experience Flows

### Flow 1: Document Seller Journey

**Step 1 - Upload Document**
- User connects Circle Wallet
- Selects document file
- Document is encrypted locally with AES-256-GCM
- Encrypted file uploads to IPFS
- User sets price (e.g., $0.50 USDC)
- Metadata stored on Arc smart contract

**Step 2 - List for Sale**
- Document appears in marketplace
- Public metadata visible: title, category, price, file size
- Content remains encrypted
- Access controlled by smart contract

**Step 3 - Earn Revenue**
- Automatic USDC payment when document is purchased
- Instant settlement (sub-second on Arc)
- 95% to creator, 5% marketplace fee
- Real-time earnings dashboard
- Withdraw anytime to wallet

### Flow 2: Human Buyer Journey

**Step 1 - Browse Marketplace**
- Search by category, price range, popularity
- View document metadata and ratings
- No account needed to browse

**Step 2 - Purchase**
- Click "Buy & Download" button
- Circle Wallet prompts for payment (e.g., $0.50 USDC)
- x402 payment executes automatically
- Payment verified on Arc

**Step 3 - Access**
- Document downloads immediately after payment
- Client-side decryption with buyer's wallet
- Document accessible forever (buyer owns it)
- Can rate/review document

### Flow 3: AI Agent Journey (Gemini)

**Step 1 - Agent Receives Task**
- User tells agent: "Find recent cybersecurity research papers under $3"
- Agent receives budget and search criteria

**Step 2 - Autonomous Search**
- Agent calls marketplace API via Function Calling
- Searches documents by category, date, price
- Retrieves list of matching documents

**Step 3 - Evaluation**
- Agent analyzes metadata: title, description, ratings
- Agent ranks documents by relevance
- Agent checks price against budget

**Step 4 - Autonomous Purchase**
- Agent selects top document
- Agent triggers x402 payment via Function Calling
- USDC payment executes automatically
- Document downloads to agent

**Step 5 - Processing**
- Agent decrypts document
- Agent extracts insights
- Agent summarizes for user
- Agent can purchase more documents if budget remains
